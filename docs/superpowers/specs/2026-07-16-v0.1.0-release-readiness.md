  `workflowkit/examples/ocr-review`。

所有 module/import path 现统一位于：

```text
github.com/eruca/goagents/...
```

发布模块的内部依赖统一为 `v0.1.0`，其 `go.mod` 不保留内部相对 `replace`。根 `go.work`
只对精确 `module@v0.1.0` 提供本地目录映射；示例模块保留相对 replace 以组合当前 checkout。

   `workflowkit`，并先做全新 consumer 验证；

   `workflowkit/agentstep`，再次做全新 consumer 验证；
3. 最后发布依赖 `mcpkit` 的 `mcpkit/officialsdk`，并验证其独立 consumer；
4. 任一层验证失败即停止，不继续发布后续 tag。
